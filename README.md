# 닥터빌 자동화 프로그램 (DVA)

닥터빌 학습 플랫폼의 반복적인 작업을 자동화하는 데스크톱 프로그램입니다.

## 주요 기능

- 🔐 **자동로그인**: 프로그램 실행 시 자동 로그인, 포인트/출석/퀴즈 상태 확인
- ✅ **출석체크**: 클릭 한 번으로 출석체크 완료
- 🧠 **퀴즈풀기**: 블로그 자동 검색으로 정답을 찾아 퀴즈 자동 풀기
- 📺 **라이브세미나**: 세미나 현황 확인, 체크박스 일괄 신청/취소, 더블클릭 입장
- 📝 **설문참여**: 저장된 정답 활용 + 자동 답변 및 제출
- 📋 **설문 문제 관리**: 설문 퀴즈 문제/정답 등록, 수정, 삭제 (전용 관리 창)
- ⚙️ **설정 시스템**: 헤드리스 모드, 로그인 후 자동 작업 실행 등 설정 가능
- 💡 **버튼 툴팁**: 각 버튼 위에 마우스를 올리면 기능 설명이 표시됩니다
- 🔄 **자동 업데이트**: GitHub에서 최신 버전을 자동으로 다운로드

## 설치 및 실행

### 🚀 자동 설치 (추천)

**처음 사용하는 경우:**

1. **`설치하기.bat`** 파일을 더블클릭하세요
2. 자동으로 다음 작업이 수행됩니다:
   - ✅ Python 설치 여부 확인 및 자동 설치 (없는 경우)
   - ✅ 필요한 패키지 자동 설치
3. 프롬프트에서 계정 정보를 입력하세요:
   - `ACCOUNT_USERNAME`: 로그인 이메일 주소
   - `ACCOUNT_PASSWORD`: 로그인 비밀번호
4. `account.bat` 파일이 자동으로 생성됩니다
5. `account.bat` 파일을 더블클릭하여 프로그램을 실행하세요

> **참고:** Python 자동 설치 시 관리자 권한이 필요할 수 있습니다.

### 📝 수동 설치

**이미 Python이 설치되어 있는 경우:**

1. **Python 설치 확인**
   - Python 3.7 이상이 설치되어 있어야 합니다
   - 설치되어 있지 않다면 [Python 공식 사이트](https://www.python.org/downloads/)에서 다운로드하세요

2. **필요한 패키지 설치**
```bash
python -m pip install -r requirements.txt
```

3. **계정 파일 생성**
   - 메모장으로 `account.bat` 파일을 생성하고 다음을 입력하세요:
   ```bat
   @echo off
   chcp 65001 >nul 2>&1
   set ACCOUNT_USERNAME=your-email@naver.com
   set ACCOUNT_PASSWORD=your-password
   start /min "" pythonw main_GUI.pyw
   ```
   - `your-email@naver.com`을 실제 이메일로 변경하세요
   - `your-password`를 실제 비밀번호로 변경하세요
   - 파일을 저장할 때 **인코딩을 ANSI로 설정**하세요

4. **프로그램 실행**
   - `account.bat` 파일을 더블클릭하세요

### 👥 다중 계정 사용 (고급)

여러 계정을 동시에 사용하려면:
- `account.bat`를 복사하여 `계정1_닥터빌.bat`, `계정2_닥터빌.bat` 등으로 이름을 바꾸세요
- 각 파일에 서로 다른 `ACCOUNT_USERNAME`, `ACCOUNT_PASSWORD`를 설정하세요
- 각 배치 파일을 더블클릭하면 독립적인 프로그램 창이 열립니다

## 사용법

1. **`account.bat`** 파일을 더블클릭하여 프로그램을 실행합니다
2. 프로그램이 자동으로 로그인하고, 포인트와 활동 상태를 확인합니다
3. 설정에 따라 **자동 작업**(출석체크, 퀴즈풀기 등)이 자동 실행될 수 있습니다
4. 각 버튼 위에 마우스를 잠시 올려놓으면 **기능 설명 툴팁**이 표시됩니다
5. 작업 진행 상황은 하단의 로그 영역에서 확인할 수 있습니다
6. 현재 상태(포인트, 출석체크/퀴즈 완료 여부)는 상단의 사용자 정보 패널에서 확인할 수 있습니다
7. 오늘의 세미나 정보는 중앙의 세미나 트리뷰에서 확인하고, **더블클릭**으로 상세보기/신청이 가능합니다

### ⚙️ 설정 변경

프로그램 내 **설정** 버튼을 클릭하여 다음 옵션을 변경할 수 있습니다:
- **브라우저 숨기기(헤드리스)**: 브라우저 창 없이 백그라운드에서 실행
- **자동 출석체크**: 로그인 후 자동으로 출석체크 실행
- **자동 퀴즈풀기**: 로그인 후 자동으로 퀴즈풀기 실행
- **자동 세미나확인**: 로그인 후 자동으로 세미나 현황 확인
- **자동 설문참여**: 로그인 후 자동으로 설문참여 실행
- **자동 세미나 신청**: 로그인 후 자동으로 세미나 신청

## 프로그램 업데이트

### 자동 업데이트 도구 사용
프로그램을 최신 버전으로 업데이트하려면:

1. **`업데이트.bat`** 파일을 더블클릭하세요
2. "업데이트 하시겠습니까?" 팝업에서 **"예"** 클릭
3. "업데이트 중입니다..." 창에서 대기
4. "업데이트가 완료되었습니다!" 팝업 확인

### 업데이트 특징
- ✅ **개인 파일 보존**: 계정 파일(.bat), chromedriver.exe, 설정 파일(.json, .ini, .log) 등은 자동으로 보존됩니다
- ✅ **안전한 백업**: 업데이트 전 자동 백업, 실패 시 자동 복원
- ✅ **간편한 실행**: 더블클릭만으로 업데이트 완료

## 주의사항

- Chrome 브라우저가 설치되어 있어야 합니다
- ChromeDriver는 폴더에 포함되어 있으며, 버전 불일치 시 자동으로 업데이트됩니다
- 프로그램을 종료하려면 창의 X 버튼을 클릭하세요
- **업데이트 중에는 프로그램을 사용하지 마세요**
- 인터넷 연결이 필요합니다 (설치, 업데이트, 자동화 실행 시)
- 계정 정보는 로컬 `.bat` 파일에 저장되며, `.gitignore`에 의해 Git에 업로드되지 않습니다

## 개발 환경

- Python 3.13+ (설치 시 자동 최신 버전 설치)
- tkinter (GUI 프레임워크)
- Selenium (웹 자동화)
- threading (비동기 처리)
